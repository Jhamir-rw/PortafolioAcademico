# Semana 14: Desarrollo Backend con Python (Flask + MySQL + Plantillas)

## 1. Introducción
En esta semana se trabajó el desarrollo backend con **Python**, revisando cómo un servidor como **Apache** puede ejecutar aplicaciones Python mediante **mod_wsgi**, y reforzando fundamentos del lenguaje: instalación, ejecución, sintaxis y buenas prácticas **PEP-8**.  

Luego se introdujo el framework **Flask** para construir aplicaciones web ligeras con rutas, plantillas y conexión a base de datos.  
También se explicó el motor de plantillas **Jinja2** para generar HTML dinámico y separar lógica (backend) de presentación (frontend).

## 2. Lo que vimos en clase
- Apache HTTP Server: request/response y entrega de contenido web.  
- Instalación portable de Apache (Apache Lounge) y configuración básica en **httpd.conf**.  
- ¿Qué es **mod_wsgi** y para qué sirve: integrar Python (Flask/Django) dentro de Apache.  
- Instalación de Python y verificación de versión (`python --version`).  
- Gestión de paquetes con **PIP** (`pip install`, `pip list`, `requirements.txt`).  
- Estándares de estilo con **PEP-8** (indentación, nombres y organización de imports).  
- Fundamentos de sintaxis Python: variables, condicionales, bucles, funciones y POO.  
- Flask: rutas con `@app.route`, request/response y `render_template`.  
- Jinja2: variables, condicionales, bucles, herencia de plantillas y filtros.
